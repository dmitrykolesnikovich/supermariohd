# README #
This is HD remake of super mario (1985).
Project was created in learning purposes only. 

Additional dependincies, tools and libraries: 
- IDE: Visual studio 2017 community(https://www.visualstudio.com/downloads)
- Libraries: SDML 2.7.0 (https://www.sfml-dev.org), tinyXML2 (https://github.com/leethomason/tinyxml2)
- Level editor: Tiled2 (http://www.mapeditor.org)
- Graphical editor: PaintNet (https://www.getpaint.net)
- Art: JinnDEvil, SuperMario All-Stars (SNES) and others free resources

Using:
- Install OpenAl (https://www.openal.org)
- Open Mario/Mario.sln file in VS2017
- Build and run project
- GL & HF
